# weather_app

Weather app with lottie animations

---

### Uses:

- *Run in terminal:*

      flutter pub get
- *Packages:*

      http: ^1.1.0
      geolocator: ^10.1.0
      geocoding: ^2.1.1
      lottie: ^2.7.0

---

### Requirements:

*Check [Open Weather Map](https://openweathermap.org) out.*
- *Get the Api Key*
- *Replace with `YOUR_API_KEY` in `lib/services/weather_service.dart`*
